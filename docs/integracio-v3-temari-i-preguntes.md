# Integració v3 del temari i preguntes

Data: 2026-05-19

S’ha refet el banc a partir de la pauta docs/pauta-refactor-temari-llica.md, del PDF base de Caldes i dels cinc informes v3 dels agents. El PDF s’ha emprat com a orientació de profunditat i estructura; les referències municipals s’han adaptat a Lliçà d’Amunt i l’Annex II s’ha mantingut en el marc 0-3 de Catalunya.

Total de preguntes integrades: **170**.
